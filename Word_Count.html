Word Count

This is a simple Python script that takes in files and counts the number of times each word is used through out the file.

The data folder that comes with the project contains some example texts, that can be used to test the program.

This program uses a map (dictionary in python) data structure to hold each word and its count in string to integers pairs. It opens a file, reads each word in the file, sanitizes the file (removes any special characters such as commas,) and then check the dictionary. If the word exists the value is incremented, if not, the value is set to 1.

The script also computes the word with the most uses, given a minimum count to consider.
