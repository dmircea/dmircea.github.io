Learning Vim


About Vim

Vim is a text-editor, derived from the orignal Vi. It is available as a GUI application, but was originally developed for the terminal. It is an open-source application that is automatically installed in most Linux operatins systems and in every copy of macOS. Additionally, it is available for download on most other OS including Windows.

Utilizing Vim can maximize productivity, since all shortcuts and key-bindings are designed in order to keep the typers hands on the keyboard, and not have them wander over a mouse or on the arrow keys. However it is also very difficult to learn how to use Vim, since memorizing all the key shortcuts must be done over the course of time.

My interest in Vim

I spent a good amount of time on the terminal with my projects, and decided that learning Vim can be helpful for tweaking the files, searching for important bits of information, especially because it is used by default in many programs (GitHub commit command will automatically open an instance Vim for the message, as an example.)

Also it seems to be pretty fun.

Vim Commands

Lasty, I made this page as my personal reminder of all the shortcuts that can be used in Vim and how they are used.

Command Use
:q Quits the current file as long as no changes have been made.
:i Enter insert mode used for typing.
:w Writes all changes to the file. Same as saving the file.
:qa Quit all files if multiple are open.
:q! or :qa! Force quits. If changes are made but not written, this will discard the changes.
w Move cursor to the beginning of the next word.
b Move cursor to the beginning of the previous word.
e Move cursor to the end of current or next word.
h j k l Moves cursor left, down, up, and right respectively.
) Moves to the begining of the next sentence, assuming English format.
( Moves to the beginning of the previous sentence, assuming English format.
ctrl+[ or Esc Exits other modes back into normal mode.
dw Delete the word from the cursor forward.
db Delete the previous word or the current word from the cursor back.
d) Delete the rest of the sentence from the cursor.
d( Delete the previous sentence or from the begining of the sentence until the cursor.
dd Delete entire line.
u Equivalent of undo command.
yw Copy the next word.
y) Copy the next sentence.
yy Copy the entire line.
p Paste from the Vim clipboard.
shift+p Paste before the word.
:/ followed by regex Allows searching using regular expressions.
