Viginere Cypher

Viginere Cypher

This program emulates the Viginere Cypher, and can be used to encrypt or decrypt passages of text, or files. The program uses a menu based interface, where the user may choose to encrypt or decrypt, a piece of text or a file. The program has no expected limitations, however it has not been tested on large files that are GigaBytes in size.

This is a project that I have started about 3 months ago. I worked on this in my free time, and attempted to make it as simple as possible. It is the first project on which I used "pylint" in order to perfect the formatting. It is still a work in progress.

This program has been tested on macOS (python 3.7.5 and 3.8.2), Xubuntu (python 3.8.1). It should work on Windows with any version of Python3.
