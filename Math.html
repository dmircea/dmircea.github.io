Mathematics program

This is a simple mathematics program that contains statistical computation. It allows the user to add numbers to a list, and compute the statistical data such as mode, or median of that list.

I originally started this project as a review to python for the first semester of graduate school, in which three of my courses were using python. It ended up growing a bit bigger than I expected, and I do plan to add a few graphical functionalities to it.

This program has been tested on macOS (python 3.7.5), Xubuntu (python 3.8.1)
