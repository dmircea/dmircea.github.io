Vector Mimic Program

    Important aspects:

  • Object-oriented programming in C++.
  • Use of the google test library.
  • Use of templates and dynamic memory allocation.
  • Contains small python script for easy compilation.

This program is a simple, different implementation of the Vector data structure from the C++ STL.

This program is not meant to be a replacement to the vector class. It is only mean to be an implementation based on my own thought process on how vectors work behind the scenes. I made this to see if I could replicate a vector's functionality with only C++, and add some other functionalities of my own.

Additionally, this project has been my own introduction to the gtest library for unit testing, on Linux.

Please note: the header file contains all the neccessary code to use the Vector_Mimic data structure. However the main.cpp utilizes the gtest library (which also uses the pthread library). Please make sure you have those libraries installed to run the test.
