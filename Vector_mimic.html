Vector Mimic Program

This program is a simple, different implementation of the Vector data structure from the C++ STL.

This program is not meant to be a replacement to the vector class. It is only mean to be an implementation based on my own thought process on how vectors work behind the scenes. I made this to see if I could replicate a vector's functionality with only C++.

This program has been tested on macOS and Xubuntu.

Download project here!
