Random maze generator

This python script creates a simple random text maze. The maze is proceduraly generated on demand using Kruskal's Algorithm. While the size of the maze is not restricted by a limit, it is suggested to keep the maze smaller than 70 by 70 characters. As a rough example, on a macOS using an Intel i5 and 8 GB RAM, a 70x70 maze was completed in approximately 20 seconds.

The algorithm used is not fully optimized. This project was created during an Introduction to Game Design course. I challenged myself to finish it in less than a week, and this project is the outcome of that challenge. Any future updates will be kept spearate from the original challenge.

This program has been tested on macOS (python 3.7.5, 3.6.8 and 3.8.2), Xubuntu (python 3.8.1) and Windows (python 3.5.7) using Python IDLE.
